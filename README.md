# sublime-telegram
A sublime text plugin for telegram

# How to use
* git clone
* ln -s /PATH/TO/SOURCE/sublime-telegram.py /PATH/TO/PACKAGES/Packages/User/sublime-telegram.py 
* (for now)
* open sublime console
* view.run_command("example")
